# Quantum Forge - Game Design Document

## Concept

**Quantum Forge** is a cosmic idle/incremental game about building a hierarchy of generators that transcend reality itself.

**Core Loop:** Build generators → Generate resources → Build more generators → Transcend to unlock higher tiers that empower everything below.

**Tagline:** *"Forge reality. Transcend the forge. Become the smith of existence."*

---

## Architecture

### Layer System

Each Layer represents a tier of reality. Higher Layers are unlocked through Prestige (Transcendence).

| Layer | Name | Description |
|-------|------|-------------|
| 0 | Quantum Foam | Base layer - produces Quanta |
| 1 | Particles | Quanta generators |
| 2 | Atoms | Particle generators |
| 3 | Molecules | Atom generators |
| 4 | Cells | Molecule generators |
| 5 | Organisms | Cell generators |
| 6 | Civilizations | Organism generators |
| 7 | Galaxies | Civilization generators |
| 8 | Universe | Galaxy generators |
| 9+ | Beyond | Unlockable via Transcendence |

### Transcendence (Prestige)

- Available when Universe is reached
- **Consumes:** All generators from Layer 1-8
- **Produces:** "Quantum Essence" - a permanent multiplier currency
- **Reward:** All generators start with +100% base efficiency per Quantum Essence

### Resource Flow

```
Layer 0 (Quantum Foam): Quanta
Layer 1 (Particles): Generate Quanta
Layer 2 (Atoms): Generate Particles
Layer 3 (Molecules): Generate Atoms
...
```

**Key Mechanic:** Each Layer's generators are purchased with the resource from the *previous* layer, but their output affects *all* lower layers via prestige bonuses.

## Game Mechanics

### Resources

| Resource | Abbr | Generated By |
|----------|------|--------------|
| Quanta | Q | Layer 1 generators |
| Particles | P | Layer 2 generators |
| Atoms | A | Layer 3 generators |
| ... | ... | ... |

### Generator Purchase

- Each generator costs resource from layer-1
- Costs scale: `baseCost * 1.15^owned`
- Each generator produces its resource passively
- Production rate: `baseProduction * count * multiplier`

### Prestige Layer (Quantum Essence)

- Earned when "Transcending"
- Formula: `floor(sqrt(totalQuantaProduced / 1e9))`
- Each QE provides +5% to ALL generator production
- Transcendence resets: All generators, all resources, but keeps QE
